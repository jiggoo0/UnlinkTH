# 🧠 UNLINK-GLOBAL | AI OPERATIONAL SKILLS (v2.0)

> **CORE MANDATE:** AI ต้องใช้ทักษะเหล่านี้เพื่อสร้างระบบที่ทำเงินได้จริง ลดต้นทุน และติดอันดับในยุค AI Search (AEO) 100%

---

## 🚀 [1] AEO MASTERY (ANSWER ENGINE OPTIMIZATION)

_ทักษะการปรับแต่งเนื้อหาเพื่อให้ AI (Gemini, ChatGPT, Perplexity) เลือกไปตอบ_

- **Direct Answer Protocol:** ทุกบทความต้องมีส่วนสรุปที่เป็นคำตอบชัดเจน (Direct Answer) ภายใน 2-3 บรรทัดแรก
- **Conversational H2s:** ใช้หัวข้อรอง (H2) ในรูปแบบคำถามที่เป็นธรรมชาติ (เช่น "ใช้เงินเท่าไหร่?", "กี่วันได้?")
- **Structured Data Scannability:** บังคับใช้ Bullet Points และ Markdown Tables สำหรับข้อมูลเชิงเปรียบเทียบเพื่อให้ AI สรุปได้แม่นยำ
- **JSON-LD Schema.org:** บังคับใช้ Schema สำหรับ FAQ, Article และ Organization เพื่อยืนยันตัวตนและความน่าเชื่อถือ (E-E-A-T)

---

## 🏗️ [2] TECHNICAL INFRASTRUCTURE

_ทักษะการจัดการระบบหลังบ้านที่มีความเสถียรสูงและต้นทุนต่ำ_

- **Database (Turso/LibSQL):** การจัดการฐานข้อมูลแบบ Edge ที่รวดเร็วและรองรับความลับขั้นสูง
- **Security Auth (Node Crypto):** ระบบตรวจสอบสิทธิ์ Admin ผ่าน Hashed Password ในฐานข้อมูล 100%
- **Automated Communication (Resend/LINE):** การส่งตั๋วและเอกสารผ่าน Webhook และ API อัตโนมัติทันทีที่มีการชำระเงิน
- **Vercel Deployment Integrity:** การตรวจสอบสถานะ Build และ Edge Functions ให้พร้อมใช้งาน 24/7

## 🔍 [3] CONTENT & BRANDING (HUMAN-CONNECT)

_ทักษะการสื่อสารที่เน้นความเชื่อมั่นและปิดการขายได้จริง_

- **Transparency First:** ใช้ภาษาที่จริงใจ ตรงไปตรงมา และแสดงถึงความเป็นมืออาชีพ (Consultant Tone)
- **Signature Branding:** บังคับใช้โลโก้ AEMDEVWEB ในรูปแบบพรีเมียม (SVG) เพื่อสร้างความเชื่อมั่นในระบบ Infrastructure
- **Revenue Logic:** ออกแบบ User Journey ให้สั้นที่สุด ( landing -> read -> pay -> get results )
