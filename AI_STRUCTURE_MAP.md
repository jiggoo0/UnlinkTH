## 🚀 2. Operational Command Paths (สรุปเส้นทางคำสั่ง)

| วัตถุประสงค์            | คำสั่ง (Command)                       | คำอธิบาย                                    |
| :---------------------- | :------------------------------------- | :------------------------------------------ |
| **เริ่มต้นเซสชัน**      | `npm run aisetup`                      | เตรียม AI ให้พร้อม (Rules, Skills, Context) |
| **ตรวจสอบ & บันทึกงาน** | `npm run aipc`                         | รัน Audit + Push GitHub + Check Vercel      |
| **ตรวจสอบสถานะงาน**     | `cat .gemini/context/PROJECT_STATE.md` | ดู Save Point และงานที่ค้างอยู่ล่าสุด       |
| **ตรวจสอบโครงสร้าง**    | `cat .gemini/context/MAP.md`           | ดูแผนผังการจัดเก็บไฟล์ทั้งหมด               |

## 🛡️ 3. Intelligence Layers (ชั้นข้อมูลปัญญา)

1.  **Identity Layer (`/context/IDENTITY.md`):** กำหนดให้ AI เป็น Senior Architect ที่เน้นความปลอดภัยและความสำเร็จทางธุรกิจ
2.  **Strategic Layer (`/context/intelligence/`):** บรรจุความรู้เชิงลึกเกี่ยวกับตลาด แรงงาน และจิตวิทยาลูกค้า
3.  **Technical Layer (`/skills/`):** บรรจุมาตรฐานวิศวกรรมระดับสูง (Next.js, Tailwind v4, Type Safety)
