# 🌐 UNLINK-GLOBAL | Technology Stack (โครงสร้างเทคโนโลยี)

โครงสร้างเทคโนโลยีนี้ถูกออกแบบมาเพื่อรองรับการทำงานแบบ **AI-Driven Development (AI เป็นผู้พัฒนาหลัก 95%)** โดยระบบต้องเอื้อให้ AI สามารถแทรกแซง สร้างฟังก์ชัน เชื่อมต่อ API และจัดการฐานข้อมูลได้อย่างมีประสิทธิภาพสูงสุดตามคำสั่งของผู้ใช้งาน

## 1. การขับเคลื่อนด้วย AI เป็นหลัก (AI-Centric Development)

- **AI Agent (Gemini CLI / Maestro):** เป็นผู้พัฒนาหลัก (95% ของงาน) รับผิดชอบในการเขียนโค้ด สร้างตรรกะ เชื่อมต่อระบบภายนอก และตรวจสอบข้อผิดพลาด โดยผู้ใช้ทำหน้าที่เป็นผู้ออกคำสั่งและกำหนดทิศทางเท่านั้น
- **Automated Validation:** มีสคริปต์ตรวจสอบความถูกต้อง (เช่น `aipc.sh`) เพื่อให้ AI รันเช็คงานของตัวเองเสมอ

## 2. ภาษาและเฟรมเวิร์กหลัก (Core Language & Framework)

- **ภาษาโปรแกรม:** TypeScript (เน้นการประกาศ Type ชัดเจนเพื่อให้ AI อ่านโค้ดและวิเคราะห์โครงสร้างได้แม่นยำที่สุด)
- **เฟรมเวิร์กหลัก:** Next.js 16 (App Router) คู่กับ React 19 (ใช้ Server Components และ Server Actions เพื่อลดความซับซ้อนให้ AI จัดการ State ได้ง่ายขึ้น)

## 3. ฐานข้อมูลและโครงสร้างพื้นฐาน (Database & Infrastructure)

- **ฐานข้อมูลหลัก:** Turso (LibSQL/D1) ฐานข้อมูลแบบ Edge SQLite ที่ AI สามารถเขียน Query สร้าง Schema และจัดการ Data Migration ผ่านคำสั่งได้อย่างรวดเร็วและปลอดภัย
- **กฎเหล็กข้อมูล:** จะไม่มีการบันทึกข้อมูลส่วนบุคคล (PII) ของลูกค้าลงในฐานข้อมูลนี้เด็ดขาด

## 4. การจัดการส่วนแสดงผล (Styling & UI)

- **CSS Framework:** Tailwind CSS 4.2.1 (Pure CSS approach)
- **แอนิเมชัน:** Framer Motion (ออกแบบให้ AI เรียกใช้งาน Component ที่เป็นเทมเพลตได้ทันที)

## 5. การเชื่อมต่อระบบภายนอก (Integrations & APIs)

- **ช่องทางการสื่อสาร:** LINE Bot SDK (ส่งลูกค้าเข้าสู่ "ช่องทางลับ")
- **การเชื่อมต่อภายนอก:** Google APIs, Vercel Analytics, และ Vercel Blob ซึ่ง AI สามารถเขียนฟังก์ชันเชื่อมต่อ (Integration) ได้ตามสถานการณ์ที่ผู้ใช้ร้องขอ
